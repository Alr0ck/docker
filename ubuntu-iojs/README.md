## partlab/ubuntu-iojs

Simple base docker image to run Node.js or Io.js applications

### Usage

#### Build

```bash
$ docker build -t partlab/ubuntu-iojs .
```

#### Download automated build

```bash
$ docker pull partlab/ubuntu-iojs
```

#### Usage

```bash
$ docker run -it --rm partlab/ubuntu-iojs iojs --version
```
